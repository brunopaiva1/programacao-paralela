3. Escreva um pseudocódigo para uma soma global estruturada em árvore. Suponha que o
número de núcleos seja uma potência de dois. 

Resposta:

supondo que p = 8


step = 1;
div = 2;

while(rank <= p){
    if(rank % div == 0){
        // Recv
        partiner = rank + step;
        sum += my_value;
    } else{
        // Send
        partiner = rank - step;
        break;
    }

    div *= 2;
    step *= 2;
}

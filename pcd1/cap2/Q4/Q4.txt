4. Um programa que obtém speedup linear é fortemente escalável? Explique sua resposta.

Resposta: Sim, pois na escalabilidade forte o tamanho do problema, ou seja n, é fixo. 
Já que esse cenário, um programa atinge um speedup linear, isso significa que o tempo 
de execução foi reduzido na proporção exata dos processadores adicionados. Ou seja, o 
trabalho fixo foi dividido com 100% de eficiência, provando que o sistema escalou 
perfeitamente.

speedup = aceleração

speedup = Tserial / Tparallel
eficiência = speedup / p

60s = 1
30s = 2
15s = 4
7.5s = 8

n = 8000 x 8000

interação = 1  --> 1.0
2 interação = 2  --> 1.0
3 interação = 4  --> 1.0
4 interação = 8 --> 1.0

6. Elabore fórmulas para o número de recebimentos e adições que o núcleo 0 realiza usando
    (a) uma soma global onde apenas o núcleo 0 realiza as adições;
    (b) uma soma global estruturada em árvore.
Faça uma tabela mostrando os números de recebimentos e adições realizados pelo nú-
cleo 0 quando as duas somas são usadas com 2, 4, 8, · · · , 1024 núcleos.

Resposta:

(A) 
Todos os cores, menos o zero enviam seus valores diretamente para o núcleo 0
que processa um por um.

Número de recebimentos: p - 1;
Número de adições: p - 1

(B)
Neste contexto o trabalho é logarítmico, isso porque a cada nível da árvore, o core 0
recebe apenas uma mensagem de um parceiro e realiza uma adição. O número de níveis da árvore 
é definido pela potência de 2.

Número de recebimentos: Log2p
Número de adições: Log2p
